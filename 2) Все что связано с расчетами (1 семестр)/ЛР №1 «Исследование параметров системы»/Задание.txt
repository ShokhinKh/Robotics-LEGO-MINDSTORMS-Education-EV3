Задание 1
Реализовать 3 алгоритма движения по границе линии с использованием одного оптического датчика (цвета/освещенности) и выполнить визуальную сравнительную оценку их качества:

Релейный с управлением по цвету (черный/белый).
Релейный с управлением по яркости отраженного света.
Адаптивный с управлением по яркости отраженного света (угол поворота пропорционален отклонению от границы линии).
Методические указания по выполнению задания 1
Во всех алгоритмах следует использовать блок рулевого управления моторами, задать одинаковые или по возможности близкие параметры, такие как скорость движения, угол поворота, заданное значение яркости границы линии и т.п. Необходимо добиться устойчивого функционирования всех алгоритмов при движении по одной и той же траектории (окружности, дуге) в обе стороны.

В качестве экспериментальных данных необходимо зафиксировать уровни яркости белого и черного фона, воспринимаемые датчиком, вычисленное на их основе значение яркости границы, и используемое в алгоритмах № 2 и № 3.


Примечание: значения яркости белого и черного фона, воспринимаемые датчиком, измеряются в нескольких точках и отыскиваются средние значения для дальнейших расчетов и использования в программе .

Задание 2
В каждом из трех алгоритмов, реализованных в соответствии с заданием 1, определить длительность одной итерации цикла и оценить влияние производительности вычислительной системы на качество управления.

Методические указания по выполнению задания 2
Для оценки производительности системы достаточно оценить длительность одной итерации - с такой периодичностью управляющая программа получает информацию от датчика, вычисляет управляющее воздействие и передает его на исполнительные механизмы (моторы).

Определить длительность итерации можно, зная общее время работы алгоритма и количество выполненных итераций.

Время работы алгоритма можно задать, выбрав соответствующий режим работы цикла, например, 5 сек. (по умолчанию). Количество итераций можно определить используя специальный выход блока цикла. Для считывания этого значения после завершения цикла рекомендуется использовать блок "Экран" в режиме отображения текста.

Учитывая, что измеряемый параметр (количество итераций) может иметь некоторый разброс, необходимо произвести не менее 5 измерений для каждого алгоритма и найти среднее значение, которое используется для дальнейшего вычисления длительности итерации. Все эксперименты необходимо выполнять в одинаковых условиях (один робот, запуск по одной и той же траектории в одном и том же направлении, одинаковые условия освещенности и т.п.).

Задание 3
Усовершенствовать алгоритм №1 и определить, насколько существенно влияние этих изменений на качество управления.

Методические указания по выполнению задания 3
Дополнить алгоритм третьим вариантом реакции на сигнал датчика цвета: если датчик не распознает ни черный, ни белый цвета, то обеспечить прямолинейное движение. Ввести в программу подсчет количества случаев выполнения этого условия и вывести его на экран.

Для подсчета количества событий (выполнение третьего варианта переключателя) необходимо в соответствующую ветвь алгоритма поместить счетчик - последовательность блоков, которые увеличивают значение некоторой переменной на 1. После завершения цикла, это значение необходимо вывести на экран вместе с общим количеством итераций (оно может измениться по сравнению с предыдущим экспериментом, т.к. включение дополнительных блоков в одну из ветвей алгоритма увеличивает время ее выполнения по сравнению с другими). Для отображения на экране одновременно двух значений можно использовать либо еще один блок "Экран", либо блок "Текст" (выполняет конкатенацию строк).

Примечания:

N3 - это количество итераций, в которых выполнялся третий вариант переключателя; Tи1, Tи2, Tи3 - значения, найденные в предыдущих экспериментах; значения в ячейках (1)-(4) рассчитываются на основе экспериментальных данных.
Данные по алгоритму № 1 (релейный, по цвету) берутся из предыдущего эксперимента, если задания 2 и 3 выполнялись в одном сеансе (на одном занятии) в одних и тех же условиях (те же робот, датчик цвета). Если задания 2 и 3 выполнялись в разных сеансах или с использованием разного оборудования, то эксперимент с алгоритмом № 1 необходимо повторить и зафиксировать его результаты в таблице к заданию 3 (в предыдущей таблице их заменять не надо!).
Длительность итерации (4) в случае выполнения третьего варианта можно вычислить из следующих отношений:

количество (1) случаев выполнения первых двух вариантов N12 = N - N3, а длительность таких итераций равна Tи1, как уже известно из первого эксперимента для алгоритма №1, отсюда можно найти суммарное время (2), затраченное на все итерации с выполнением первых двух вариантов;
оставшееся время (3), которое приходится на остальные N3 итераций с третьим вариантом ветвления, можно определить из общего времени работы алгоритма Tр за вычетом суммарного времени (2), затем можно найти длительность итерации (4).
